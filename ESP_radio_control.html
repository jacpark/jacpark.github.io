Why are we doing this lab?

We are using the ESP32 as a server to allow our computers to issue commands. If the command is recived a corrisponding LED will light up. This serves as practice for how we will operate our drones once they are ready to go.

Materials:

Arduino IDE with ESP 32 library

ESP 32

Breadboard

LED lights x 8

220ohm resistor

Jumper wires

Active internet connection

Lab Procedure

1.Gather all materials.

2.Set up Arduino with ESP 32 library Click Here

3.Set up breadboard with ESP, LEDs, and resistor

4.Load example sketch and test ESP32

5.Add to sketch to make it work with all LED's and commands

6.Test and revise until working

Results

Up:Orange LED

Right:Yellow LED

Clockwise:Green LED

Down, Left, Reverse, & Counter Clockwise:White LED opposite axis color
